# FHEVM Example Template

This is the base template for creating standalone FHEVM examples.

## Installation

```bash
npm install
```

## Compilation

```bash
npm run compile
```

## Testing

```bash
npm run test
```

## Customization

Replace the contract in `contracts/FHETemplate.sol` with your own example contract and update tests accordingly.

## Deployment

```bash
npm run deploy --network sepolia
```
